CoRAWeb

CoRAWeb is a set of REST services to be used from various augmentation plugin

While the augmentations facilities are implemented, you can use these shortcuts

  • You can experiment with the services using the explorer
  • You can drag this bookmarklet to your toolbar, to collect items. It will use existing templates. Only the first match will be used to extract an item. It currently works with Youtube videos, Slideshare presentations, and Mendeley research papers
